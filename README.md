# Sintese Hero

Landing hero em Next.js para a marca Sintese, recriando a referencia visual de automacao SDR com IA.

## Stack

- Next.js 16 App Router
- React 19
- Tailwind CSS 4
- Lucide React

## Scripts

```bash
npm run dev
npm run lint
npm run build
```

Abra `http://localhost:3000` depois de iniciar o servidor local.

## Estrutura

- `src/app/page.tsx`: hero principal e componentes estaticos.
- `src/app/globals.css`: tokens visuais, layout, motion e responsividade.
- `public/assets/hero`: imagens usadas na composicao.
- `System rules`: referencias e diretrizes de design usadas na recriacao.
