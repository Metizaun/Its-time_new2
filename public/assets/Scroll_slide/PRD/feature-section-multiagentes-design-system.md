# FEATURE SECTION — MULTIAGENTES ESPECIALIZADOS

---

# CONCEITO DA SEÇÃO

Esta seção NÃO deve parecer:
- um card de feature comum
- dashboard SaaS
- chatbot visual
- interface genérica

Ela deve parecer:
> uma infraestrutura operacional inteligente coordenando especialistas autônomos.

A sensação precisa ser:
- estratégica
- silenciosamente poderosa
- modular
- arquitetural
- premium
- precisa

---

# OBJETIVO VISUAL

Transmitir:
- orquestração inteligente
- múltiplos agentes operando juntos
- sistema autônomo
- inteligência distribuída
- coordenação operacional

A seção deve comunicar:
> “não existe apenas uma IA.”
>
> Existe um ecossistema comercial operando em conjunto.

---

# DIREÇÃO VISUAL

## Estética
Mistura de:
- OpenAI
- Linear
- Apple
- Infraestrutura neural
- Swiss editorial
- Tactical minimalism

---

# ESTRUTURA DA COMPOSIÇÃO

## Layout
Divisão:
- lado esquerdo → narrativa editorial
- lado direito → arquitetura operacional

---

# FUNDO

## Base
Preto profundo:
- #000000
- #050505

## Fundo deve conter:
- gradiente extremamente suave
- círculos pontilhados orbitais
- partículas modulares discretas
- grids invisíveis
- glow mínimo vermelho

---

# NÚCLEO CENTRAL

## Elemento principal
Cubo translúcido futurista.

Características:
- vidro técnico
- bordas suaves
- glow vermelho interno
- símbolo central luminoso
- profundidade minimalista

## Sensação
O cubo representa:
> o núcleo operacional da inteligência.

---

# CÍRCULOS AO REDOR DO CUBO

## Estrutura
- círculos pontilhados concêntricos
- extremamente discretos
- laranja/vermelho suave

## Função visual
Representar:
- processamento
- comunicação
- propagação operacional
- inteligência distribuída

Nunca parecer:
- radar militar
- sci-fi exagerado

---

# CONEXÕES

## Linhas
- finas
- técnicas
- discretas
- arquiteturais

## Nodes
- pequenos quadrados brancos
- minimalistas
- organizados

---

# AGENTES

## Estrutura
Cada agente deve ficar dentro de:
- card preto
- borda fina
- glow mínimo
- tipografia clean

## Layout dos cards
- ícone minimalista
- nome do agente
- indicador online verde
- muito espaço interno

---

# AGENTES DA SEÇÃO

- Agente SDR
- Agente Qualificador
- Agente Follow-up
- Agente Nutrição
- Agente Reativação
- Agente Closer

---

# POSICIONAMENTO DOS AGENTES

Distribuição radial.

Cada agente conectado diretamente ao núcleo central.

A composição deve parecer:
> uma arquitetura neural operacional.

---

## Headline
Grande.

Multiagentes

# SOMBRAS E GLOW

## Glow
Muito sutil.

Nunca:
- neon
- cyberpunk
- gamer

O glow deve parecer:
> energia operacional silenciosa.

---

# ESPAÇAMENTO

A composição precisa respirar.

Muito espaço negativo.

Nada apertado.

Nada poluído.

---

# MOTION DESIGN

## Movimento ideal
- partículas orbitando lentamente
- glow pulsando minimamente
- linhas respirando
- círculos movimentando suavemente

Tudo:
- lento
- elegante
- técnico

---

# NÃO USAR

## Proibido
- hologramas
- excesso de blur
- UI exagerada
- dashboards poluídos
- azul neon
- cyberpunk
- efeitos sci-fi hollywood
- glassmorphism exagerado

---

# SEMPRE USAR

- modularidade
- arquitetura visual
- grids invisíveis
- contraste refinado
- silêncio visual
- profundidade mínima
- composição editorial
- inteligência operacional

---

# SENSAÇÃO FINAL

A seção inteira deve parecer:

> um sistema comercial autônomo operando em tempo real.

E nunca:

> uma interface bonita de chatbot.
